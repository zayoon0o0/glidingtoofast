# GlidingTooFast

A simple Fabric server-side mod to warn players if they are going too fast with the Elytra.

### How to build
1. ```git clone https://github.com/zayoon0o0/glidingtoofast.git```

2. ```cd glidingtoofast```

3. ```./gradlew build```
